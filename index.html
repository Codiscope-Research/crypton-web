How Crypton works

Step 1
Install Crypton
To get started begin by installing Crypton locally via npm, the Node.js package manager
npm install -g crypton
crypton demo
Step 2
Develop application
The developer writes the application with HTML, CSS & JavaScript and can test it in the browser at any time. The development process is similar to any web page development.
Step 3
Deploy application
The developer deploys the app to multiple targets (Android, iOS & browser extensions). The deployed app talks to the Crypton server over HTTPS.
